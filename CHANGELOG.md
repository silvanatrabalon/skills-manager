# Changelog

## [1.0.0] - 2026-04-21

### Added
- Tree view of installed skills organized by scope (Global / Project)
- One-click install, update, and remove with inline buttons
- Automatic update detection via GitHub API (every 2 hours)
- Multi-repository support (GitHub, GitLab, local paths)
- Scope selection on install: Project only, Global, or Both
- Agent selection on install (Cursor, GitHub Copilot, Claude Code, etc.)
- Skill details panel with formatted webview
- Lock file enrichment with `skillFolderHash` for update tracking
- Reads skill descriptions from `SKILL.md` frontmatter
